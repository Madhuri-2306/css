Cascading Style Sheets

What is CSS?

CSS stands for cascading style sheets.It is a Language use to style a Web page.CSS describes how HTML elements are to be displayed on screen, paper, or in other media In simple words, flowing different styles to the html element.css saves a lot of work.

How to insert CSS?

  • Inline styling
  • Internal styling
  • External styling

CSS Selectors:

A CSS selector selects the HTML element(s) you want to style.Selectors are five types.

Simple selectors:

  1. Element selector
  2. Id selector
  3. Class selector
  4. Universal selector
  5. Grouping selector

1.Element selector:

The element selector selects HTML elements based on the element name.

2.Id selector:

Id selector is used to select one unique element.It is represented by '#'.

3.Class selector:

Class selector can have multiple values with space seperated.It is represented by '.'.

4.Universal selector:

The universal selector (*) selects all HTML elements on the page.

5.Grouping selector:

The grouping selector selects all the HTML elements with the same style definitions.To group selectors, separate each selector with a comma.
